# Language Emergence Lab Dashboard

Minimalist dashboard for visualizing experiment results and training metrics.

## Setup

```bash
source venv/bin/activate
pip install -e .
```

## Run

```bash
cd dashboard
python main.py
```

Open `http://localhost:5000` in your browser.

## Features

- **Overview**: Aggregate statistics and training progress
- **Experiments**: Browse and filter experiment results
- **Training**: Detailed metrics and loss curves

## Design

Modern, clean interface with sidebar navigation:
- Font: Space Grotesk
- Colors: Indigo (#6366F1), Emerald (#059669), Orange (#F97316)
- Layout: Sidebar navigation with responsive grid-based content
- Subtle shadows and smooth transitions
- Mobile-friendly with collapsible sidebar
