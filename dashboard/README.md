# Language Emergence Lab Dashboard

Minimalist dashboard for visualizing experiment results and training metrics.

## Setup

```bash
source venv/bin/activate
pip install -e .
```

## Run

```bash
cd dashboard
python main.py
```

Open `http://localhost:5000` in your browser.

## Features

- **Overview**: Aggregate statistics and training progress
- **Experiments**: Browse and filter experiment results
- **Training**: Detailed metrics and loss curves

## Design

Bauhaus-inspired interface with flat colors and geometric precision:
- Font: Space Grotesk
- Colors: Black, Red (#E63946), Blue (#457B9D), Yellow (#F4A261)
- Layout: Grid-based with sharp edges
